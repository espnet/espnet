# ACESinger-KiSing-V2 RECIPE

This is the recipe of ACESinger-augmented KiSing-V2 recipe

You should download the original and augmented KiSing dataset and make sure the file structure is like this:

```
KISING
├── 421
│   ├── ...
├── 422
│   ├── ...
│   ...
├── 443
│   ├── ...
├── original
│   ├── 421_all.wav
│   ├── ...
```
