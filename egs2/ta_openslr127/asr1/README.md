# IISc-MILE Tamil ASR Corpus (OpenSLR 127) Results

End-to-end ASR recipe for the [IISc-MILE Tamil ASR Corpus](https://www.openslr.org/127/):
~150 h of read Tamil speech (sources: MILE, ISTL, MICI), 16 kHz / 16-bit / mono PCM.

| set        | utterances | description                                                 |
| ---------- | ---------- | ----------------------------------------------------------- |
| `train_ta` | 69,957     | official train, minus dev speakers (after length filtering) |
| `dev_ta`   | 7,329      | speaker-disjoint 10% held out from train                    |
| `test_ta`  | 12,087     | official test set                                           |

**Model:** E-Branchformer encoder + Transformer decoder, joint CTC/attention
(`ctc_weight=0.3`), raw log-mel frontend, BPE-1000 (unigram), no external LM.
The training config is adapted from `egs2/librispeech_100/asr1`.
